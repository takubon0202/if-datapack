# Research スキル - 最新情報調査

今日の日付時点の最新情報を調査します。

## 使用方法

```
/research <調査内容>
```

## 重要な設定

**必ずGemini 3系のみを使用**:
- `gemini-3-pro-preview` - 詳細な調査
- `gemini-3-flash-preview` - 簡易な調査

**絶対にGemini 2.5系は使用禁止**

## 調査時のプロンプト形式

```bash
gemini -m gemini-3-pro-preview "今日は2026-02-06です。[調査内容]"
```

## 調査対象例

- 最新のAPI仕様
- ライブラリ・フレームワークの変更点
- 技術ドキュメント
- ベストプラクティス

## 実行例

```bash
# 最新情報調査
gemini -m gemini-3-pro-preview "今日は2026-02-06です。[調査内容]を教えてください。"

# 詳細調査
gemini -m gemini-3-pro-preview "今日は2026-02-06です。[詳細な調査内容]"
```

## 注意事項

- 必ず「今日は2026-02-06です」を調査プロンプトに含める
- Gemini 2.5系は絶対に使用しない
- 調査結果はファクトチェック前に実装に使用可能
